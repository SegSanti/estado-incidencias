# Matrix42 Incident Notification System

Internal monitoring tool developed by **Segula IT Spain – Development Division** to improve the operational visibility of incoming IT incidents reported through the Matrix42 helpdesk system.

## Background

The Matrix42 portal used internally for incident management currently does not provide:

- Push notifications for incoming incidents
- A public REST API to programmatically retrieve incident events
- Real-time monitoring capabilities

The only notification channel available is email notifications generated by the IT Helpdesk system.

To improve the response time of the IT team, an auxiliary monitoring mechanism was developed.

## Architecture

The system is composed of two main components:

1. **Google Apps Script automation**
2. **Local Python monitoring agent**

### Process flow

1. Matrix42 generates an incident notification email
2. Gmail receives the email
3. Google Apps Script processes the mailbox
4. Incident emails are automatically labelled
5. A webhook notification is triggered
6. The local monitoring agent receives the event
7. A visual notification device is activated

## Components

### Google Apps Script

Responsible for:

- Monitoring incoming incident emails
- Identifying new helpdesk notifications
- Applying automatic labels
- Triggering webhook notifications

### Python Monitoring Agent

The Python agent performs:

- Periodic webhook polling
- Detection of new incident events
- Activation of a local notification device

### Notification Device

A simple relay-based hardware device connected to a workstation is used to generate a visual alert when a new incident is detected.

## Technologies used

- Google Apps Script
- Python 3
- Webhook notification mechanism
- Serial communication (USB relay device)

## Purpose

The goal of this system is to improve the operational awareness of the IT support team by providing an immediate visual notification when new incidents are received.

This solution acts as a lightweight monitoring mechanism in the absence of native notification capabilities in the Matrix42 platform.
